## ?? Proje Hakkýnda

**Gazi Üniversitesi Hastanesi Web Uygulamasý**, Gazi Üniversitesi Hastanesi'nin tüm dijital hizmetlerini tek bir çatý altýnda toplayan, modern ve kullanýcý odaklý bir web platformudur. Proje, **staj çalýţmasý** kapsamýnda sýfýrdan geliţtirilmektedir.

Uygulama; hastalarýn online randevu almasýndan test sonuçlarýna eriţmesine, borç sorgulamadan kurumsal bilgilere ulaţmasýna kadar geniţ bir hizmet yelpazesi sunmaktadýr. Ayrýca hastane personeli için kapsamlý bir **Admin Paneli** ile yönetim operasyonlarý da desteklenmektedir.

### ?? Proje Amacý

- Hastane hizmetlerini dijitalleţtirmek ve tek noktadan eriţim sađlamak
- Hasta deneyimini iyileţtirmek ve bekleme sürelerini azaltmak
- Hastane yönetimine iç operasyonlar için modern bir dashboard sunmak
- Kurumsal kimliđi profesyonel bir web arayüzü ile yansýtmak

## ?? Kurulum

### Ön Gereksinimler

- [.NET 7 SDK](https://dotnet.microsoft.com/download/dotnet/7.0) veya üzeri
- [Visual Studio 2022](https://visualstudio.microsoft.com/) (önerilen) veya VS Code
- Git

### Adýmlar

**1. Projeyi klonlayýn:**
```bash
git clone https://github.com/Hasretozdemir/GaziHastanesi.git
cd GaziHastanesi
```

**2. Bađýmlýlýklarý yükleyin:**
```bash
cd GaziHastane
dotnet restore
```

**3. Uygulamayý çalýţtýrýn:**
```bash
dotnet run
```

**4. Tarayýcýda açýn:**
```
https://localhost:5001
```

> ?? **Visual Studio** kullanýyorsanýz `.sln` dosyasýný açýp `F5` ile dođrudan çalýţtýrabilirsiniz.

## ??? Admin Paneli

Admin paneline `/Admin` URL'i üzerinden eriţilir. Panel, **Glassmorphism** tasarým diliyle geliţtirilmiţ olup ţu özellikleri içerir:

- ?? **Gösterge Paneli** – Toplam doktor, günlük randevu, aktif bölüm sayýsý, sistem yükü
- ?? **Akýllý Arama** – Hasta, doktor veya veri arama
- ?? **Aktif Hekim Tablosu** – Sistemdeki hekimlerin anlýk durumu
- ?? **Dark Theme** – Koyu tema ile profesyonel yönetim deneyimi
- ?? **Sidebar Navigasyon** – Gösterge paneli, doktorlar, poliklinikler

> ?? **Not:** Admin paneli ţu anda geliţtirme aţamasýndadýr. Kimlik dođrulama (authentication) ve veritabaný entegrasyonu henüz eklenmemiţtir.
